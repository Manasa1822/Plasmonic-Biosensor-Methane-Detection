# Plasmonic-Biosensor-Methane-Detection

## Overview
This project presents a Kretschmann configuration-based plasmonic biosensor for methane detection using Graphene, Blue Phosphorene, Silver (Ag), and As₂S₃.

## Technologies Used
- MATLAB
- Plasmonic Biosensor
- Graphene
- Blue Phosphorene
